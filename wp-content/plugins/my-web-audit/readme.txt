== Description ==

Provide technical information for the website to My Web Audit app in a very secure fashion  with the help of public and private key pairs so that the data exported can never be decrypted by any third party application. Data gathered includes: 
    - Current WordPress version installed
    - Site timezone
    - No. of active and inactive themes installed on the website
    - No. of active and inactive plugins installed on the website
    - Count of users on the Administrator role
    - Status of the availability of default WP "Sample Page", "Hello world!" post and "default comments"

== Installation ==

1. Download My Web Audit Plugin.
1. Visit Plugins > Add New
1. Click on upload plugin button.
1. Click on browse button and select plugin zip file. After file select click on Install Now button.
1. Click on Activate Plugin button.
